# Hotelio

Hotelio is app that showing best hotel. So you can staycation with chill

## Features

- Showing list hotel
- Detail about hotel


## Tech

Hotelio App uses a number of open source projects to work properly:

- Material
- ViewBinding
- Recyclerview
- Glide
